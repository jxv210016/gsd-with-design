  <action>
Create `install.ps1` at the project repo root. Target PowerShell 5.1+ (ships with Windows 10/11).

**Script structure:**

1. **Execution policy handling:** At the top, check if execution policy allows script execution. If restricted at process scope, set process-scoped bypass: `Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass -Force`. This only affects the current process, not the system. If this fails, print clear error: "Run with: powershell -ExecutionPolicy Bypass -File install.ps1"

2. **Banner:** Write-Host "GSD with Design -- Overlay Installer v1.0.0"

3. **Helper function `Get-DesignFileHash`:** Takes a file path, returns lowercase SHA-256 hash using `(Get-FileHash -Path $FilePath -Algorithm SHA256).Hash.ToLower()`.

4. **Source directory detection:** Script runs from repo root. Test for `.claude\commands\gsd\design-thinking.md` and `workflows\design\ui-design.md`. If missing, error: "Run this script from the gsd-with-design repository root."

5. **GSD installation detection (R6.5):** Search for `get-shit-done\VERSION` in:
   - Local: `.\{.claude,.config\opencode,.opencode,.gemini}`
   - Global: `$HOME\{.claude,.config\opencode,.opencode,.gemini}`

   Collect all found into an array. Zero: error "No GSD installation found." One: use it. Multiple: list numbered, prompt `Read-Host "Select installation"`, default to first.

6. **Existing design installation check:** If `design-version.json` exists at target, read it with `ConvertFrom-Json`. For each file, compare current hash to stored hash. If mismatches found, prompt: "Customized design files detected. Back up? [Y/n]" -- if yes, copy to `get-shit-done\design-backup-{timestamp}\`.

7. **Path rewriting:** The patched command shims contain `@/Users/jayvanam/.claude/` paths. For each patched file (new-project.md, discuss-phase.md, plan-phase.md, update.md in commands/gsd/, and new-project.md, discuss-phase.md, plan-phase.md in workflows/), read content, replace `@/Users/jayvanam/.claude/` with the install dir path using forward slashes (GSD @ references use forward slashes even on Windows): `@$($InstallDir -replace '\\','/')/`. Write to target.

8. **File copy:** Create `get-shit-done\workflows\design\` dir if missing (`New-Item -ItemType Directory -Force`). Copy design-only files directly with `Copy-Item`. Copy patched files through path rewriting (read, replace, write).

9. **design-version.json generation:** After all files installed, build a hashtable/PSCustomObject with version, schema_version, installed_at (Get-Date -Format o), gsd_base_version (from VERSION file), and files map (each file path -> "sha256:HASH"). Convert to JSON with `ConvertTo-Json -Depth 3` and write to `{INSTALL}\get-shit-done\design-version.json`. Use forward slashes in the file keys (matching the schema from 06-01-PLAN).

10. **Verification and summary:** Count installed files, verify all 16 exist. Print: files installed, install location, GSD base version.

11. **Success:** "GSD with Design installed successfully! Restart Claude Code to use design commands."
