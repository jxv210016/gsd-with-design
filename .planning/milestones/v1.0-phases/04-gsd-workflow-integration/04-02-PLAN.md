```markdown
<!-- GSD-DESIGN-START -->
<step name="design_detection">
## Design Detection and Agent Orchestration

**Guard clause:** Check if `.planning/DESIGN.md` exists.
- If DESIGN.md does NOT exist: Log "No DESIGN.md found -- skipping design detection." Continue to confirm_creation.

**Run UI detection:**

Execute the detection algorithm from the ui-detection workflow:

@workflows/design/ui-detection.md

Steps:
1. Get phase section text: `node "$HOME/.claude/get-shit-done/bin/gsd-tools.cjs" roadmap get-phase "${PHASE}"`
2. Read CONTEXT.md text from `${phase_dir}/${padded_phase}-CONTEXT.md`
3. Concatenate phase section + CONTEXT.md into SCAN_TEXT
4. Apply priority-ordered detection:
   a. Check for manual override markers (`<!-- ui-phase -->` or `<!-- no-ui -->`) in ROADMAP phase section
   b. Check for negative keyword dominance (migration, CLI, API-only, etc.)
   c. Count positive keyword category matches across 6 categories with 2+ threshold
5. Capture result: IS_UI, DETECTION_METHOD, MATCHED_CATEGORIES

**If IS_UI=false:**
Log: "Phase ${PHASE} is not a UI phase (${DETECTION_METHOD}). Skipping design agents."
Continue to confirm_creation.

**If IS_UI=true:**
Log: "UI phase detected (${DETECTION_METHOD}: ${MATCHED_CATEGORIES}). Spawning design agents..."

Determine REFRESH_STACK:
- If this is the first UI phase in the project (no prior {phase}-UI.md files exist), set REFRESH_STACK=true
- If stack conventions have changed since last run, set REFRESH_STACK=true
- Otherwise REFRESH_STACK=false

Execute the orchestration workflow:

@workflows/design/orchestrate-design.md

Pass context:
- PHASE_NUMBER=${PHASE}
- PHASE_NAME=${phase_name}
- PHASE_GOAL=(from ROADMAP.md phase section)
- PHASE_DIR=${phase_dir}
- CONTEXT_PATH=${phase_dir}/${padded_phase}-CONTEXT.md
- REFRESH_STACK=${REFRESH_STACK}

The orchestrator will:
1. Run stack-conventions gate (creates/refreshes STACK.md if needed)
2. Spawn 3 design agents in parallel (ui-design, ux-design, motion-design)
3. Collect results with retry-once for failures
4. Synthesize into {phase}-UI.md with conflict resolution

After orchestration completes:
- Log: "Design guidance written to ${phase_dir}/${padded_phase}-UI.md"
- Set DESIGN_UI_CREATED=true (used by confirm_creation and git_commit)

Continue to confirm_creation.
</step>
<!-- GSD-DESIGN-END -->
```
